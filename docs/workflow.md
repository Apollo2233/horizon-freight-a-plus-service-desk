# Troubleshooting Workflow Example

## Scenario: Laptop Not Powering On

### Ticket Information
- Category: Hardware
- Severity: Medium
- User Impact: Single User

---

### Step 1: Identify the Problem
User reports device will not power on. No LED indicators active. No response when pressing power button.

---

### Step 2: Establish Theory
Possible causes:
- Faulty power adapter
- Dead battery
- Motherboard failure

---

### Step 3: Test the Theory
Tested alternate power adapter.
Device powered on successfully.

---

### Step 4: Plan of Action
Replace faulty power adapter.
Verify system stability.

---

### Step 5: Verify Full Functionality
Device boots into Windows.
User confirms normal operation.

---

### Step 6: Document Findings
Root Cause: Faulty power adapter
Resolution Time: 18 minutes
Status: Closed
