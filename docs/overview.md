# Project Overview

The Horizon Freight A+ Service Desk project simulates a real-world IT helpdesk environment aligned with CompTIA A+ troubleshooting methodology.

This system models:

- Ticket intake and categorization
- Structured troubleshooting using the 6-step CompTIA method
- Root cause analysis
- Resolution documentation
- Operational metrics tracking

The goal is to demonstrate practical understanding of real-world IT support workflows beyond certification theory.
